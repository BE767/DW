CURSOS

GIT

Git es un sistema de control de versiones distribuido, lo que significa que un clon local del proyecto es un repositorio de control de versiones completo. Estos repositorios locales plenamente funcionales permiten trabajar sin conexión o de forma remota con facilidad. Actualmente se trabaja con el para poder facilitar el trabajo del desarrollador para programar y conectarse con servidores remotos,el poseer este conocimiento asegura el tiempo de creacion de la pagina web.

CSS

CSS es utilizado para diseñar y dar estilo a las páginas web, por ejemplo, alterando la fuente, color, tamaño y espaciado del contenido, dividirlo en múltiples columnas o agregar animaciones y otras características decorativas. Ayudara a personalizar cada una de las paginas para satisfacer los requerimientos del cliente

HTML

Lenguaje básico para la creación de una página web. Es un lenguaje muy sencillo compuesto de elementos, que se pueden aplicar a piezas de texto para darles un significado diferente en un documento, gracias a el crearemos el esqueleto de su página web (texto, informacion etc.) para posteriormente complementar con Bootstrap y Css

Bootstrap

Bootstrap es un marco de desarrollo front-end gratuito y de código abierto para la creación de sitios web y aplicaciones web. Diseñado para permitir el desarrollo receptivo de sitios web móviles,con esto crearemos un diseño creativo y dinamico par cumplir con cada uno de los requerimientos propuestos y entregar un proyecto de calidad
